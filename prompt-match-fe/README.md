# getting started

推荐使用pnpm作为包管理工具

如从未安装过pnpm，可先通过npm进行安装

```bash
$ npm install -g pnpm
```

完成pnpm安装后，安装项目依赖包

```bash
$ pnpm i
```

启动项目

```bash
$ pnpm dev
```
